# Smoke Test Checklist

- Dashboard loads.
- Source Manager accepts a source.
- Manual crawl creates a `CrawlRun`.
- Analyze Pending marks articles processed and creates leads or exclusions.
- Lead detail opens and status buttons work.
- A/B lead draft generation creates email and LinkedIn drafts.
- CSV and Excel-readable exports download.
- Settings shows DB counts and backup action.
- Settings reset clears collected data while keeping sources.
